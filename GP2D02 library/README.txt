This GP2D02 library is for interfacing with the Sharp GP2D02 sensor and the arduino.  

To install it, put the GP2D02 folder into the library folder of your arduino software.

This software is free to use in your own software, but please do not steal it as your own.  Simply acknowledge me.  Thanks a lot.
